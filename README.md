# 💥 爆炸模拟器

基于浏览器的 3D 物理沙盒破坏游戏。放置炸药、喷射火焰、召唤黑洞、组建火柴人军团——在低多边形城市中尽情释放破坏欲。

## 演示

```
🔫 22种道具 × 5大分类 × 实时物理破坏 × 打分系统
```

## 特性

### 武器库

| 分类 | 道具 | 玩法 |
|------|------|------|
| 💣 爆炸类 | TNT、C4、硝酸甘油、原子弹、遥控炸弹、地雷、集束炸弹 | 点击放置 → 一键引爆 → 观看冲击波摧毁一切 |
| 🔥 喷射类 | 火焰喷射器、冰冻喷射、粒子喷射 | **按住鼠标持续施放**：灼烧建筑 2s 起火、冰冻减速、粒子分解 |
| 🌀 物理装置 | 黑洞装置 | 全屏吸入所有物体 → 核心摧毁 → 霍金辐射爆发抛射 |
| 🎆 特殊类 | 燃烧弹、烟雾弹、闪光弹、电磁脉冲 | 区域性持续效果，EMP 伴随屏幕闪烁 |
| 🏗️ 建造类 | 建筑、车辆、树木、沙袋、路障、🏕️兵营 | 自定义场景布局，兵营持续产出火柴人 |

### AI 火柴人军团

- 兵营拖放即产兵（无限量），默认每 2s 一个
- 平日自由走动，爆炸靠近时惊恐加速逃跑
- HP 血量 + 冲击伤害判定，击杀 +100 分
- HUD 实时显示存活数量 👤

### 打分系统

每次引爆计算三维度得分：
- **摧毁分**：建筑 100×高度 + 车辆 200 + 树木 50
- **波及分**：冲击波范围内被推动物体 ×10
- **连锁分**：碎片撞击其他建筑产生二次破坏 +50

引爆位置飘字动画 + HUD 累计总分 + localStorage 最高分持久化。

### 物理破坏

- Cannon-es 真实刚体物理：冲击波衰减、建筑碎裂成数十块碎片
- 碎片飞行中撞击其他建筑 → 连锁倒塌
- 黑洞 100% 全屏吸入 → 物体接触核心瞬间粉碎
- 喷射武器暴露追踪 → 持续施放触发点燃/冰冻/分解

### 视觉特效

- 每种爆炸独特的粒子特效 + 蘑菇云 + 冲击环
- 黑洞三阶段：吸积盘旋转 → 螺旋粒子流 → 霍金辐射白闪光
- 引力透镜屏幕扭曲 (WebGL RenderTarget + 自定义 Shader)
- 屏幕震动 + EMP 全屏闪白

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 浏览器打开 http://localhost:5173
```

## 操作

| 操作 | 方式 |
|------|------|
| 选择道具 | 左侧面板点击卡片 或 直接拖拽到场景 |
| 放置 | 点击场景地面 |
| 引爆 | 面板"引爆"按钮 或 空格键 |
| 喷射武器 | 选中后**按住鼠标**持续喷射 |
| 旋转视角 | Q/E 键 或 右键拖拽 |
| 缩放 | 鼠标滚轮 |
| 重置场景 | 面板"重置"按钮 或 R 键 |
| 遥控引爆 | 1/2/3 键分组引爆 |

## 技术栈

[Three.js](https://threejs.org/) 渲染 · [Cannon-es](https://github.com/pmndrs/cannon-es) 物理 · TypeScript · Vite

低多边形风格，等距正交相机，支持阴影与雾效。
