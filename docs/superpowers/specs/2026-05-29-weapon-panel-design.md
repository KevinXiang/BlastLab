# 武器库系统设计

## 概述

将现有 4 种爆炸物扩展为完整的沙盒武器库，包含爆炸类、特殊类和建造类共 16 种工具。通过左侧可收起面板选择武器，点击或拖拽放置到 3D 场景。

## 面板布局

```
┌──────────────┐
│ [收起 ◀]     │
│              │  ← 300px 宽，半透明黑底 rgba(0,0,0,0.75)
│ ── 爆炸类 ── │
│ TNT桶   C4   │  ← 2列网格，每个卡片含 emoji 图标 + 名称
│ 硝酸甘油 原子弹│
│ 遥控炸弹 地雷 │
│              │
│ ── 特殊类 ── │
│ 燃烧弹 烟雾弹 │
│ 闪光弹       │
│              │
│ ── 建造类 ── │
│ 建筑    车辆  │
│ 树木    沙袋  │
│ 路障         │
└──────────────┘
```

## 交互

- **收起状态**：左侧边缘 24px 竖条标签 `[武]`，点击展开
- **展开状态**：面板滑出 300px（CSS transition 0.2s）
- **选中**：单击卡片高亮，再次点击取消选中
- **拖拽放置**：dragstart 设置 data，drop 在 canvas 上方处理
- **快捷键**：按 ` 键（反引号）切换面板展开/收起

## 底部工具栏调整

移除建造类卡片（建筑/车辆/树木），保留：

```
[硝甘] [TNT] [C4] [核] ┃ [引爆] [重置]
```

武器库面板出现后，底部仅为快捷栏。

## 遥控炸弹分组引爆

```
placeRemoteBomb(position, group: 0|1|2)
  → 存入 remoteBombs[group][]
  → 按键 1 → detonateGroup(0)
  → 按键 2 → detonateGroup(1)
  → 按键 3 → detonateGroup(2)
```

## 地雷触发

```
每一帧 animate():
  for each mine in mines:
    for each body in world.bodies (排除地面):
      if distance(body, mine) < 1.5m 且 body.mass > 0:
        detonateMine(mine)
        break
```
