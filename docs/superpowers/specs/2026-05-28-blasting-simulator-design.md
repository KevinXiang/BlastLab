# 爆炸模拟器设计文档

## 概述

一款基于浏览器的 2.5D 等距视角物理沙盒游戏。玩家在低多边形风格的中尺度城市街区中放置各类爆炸物，观察冲击波摧毁建筑、掀翻车辆的破坏效果。提供自由沙盒和关卡挑战两种模式。

## 技术栈

| 层 | 技术 |
|---|---|
| 渲染引擎 | Three.js（正交相机等距投影、低多边形风格） |
| 物理引擎 | Cannon-es（刚体碰撞、重力、爆炸冲击力） |
| 构建工具 | Vite |
| 语言 | TypeScript |

## 架构

```
Game Loop (requestAnimationFrame)
  ├── Physics (Cannon-es)     — 刚体碰撞、重力、冲击波
  ├── Renderer (Three.js)      — 正交相机、粒子系统、low-poly 材质
  ├── Scene Manager            — 建筑管理、碎片追踪、爆炸队列
  ├── Game Mode Controller     — 沙盒模式 / 关卡模式
  └── UI Layer (DOM Overlay)   — 工具栏、状态栏、菜单
```

**关键设计原则：**
- 物理与渲染完全分离 — Cannon-es 计算位置/速度，Three.js 根据物理状态渲染
- 建筑由大块刚体组成，爆炸后替换为小块碎片刚体
- 碎片落地减速后回收刚体，转换为粒子淡出

## 爆炸物类型

| 属性 | 硝酸甘油 | TNT | C4 | 原子弹 |
|---|---|---|---|---|
| 爆炸半径 | 小 (3m) | 中 (8m) | 中 (6m) | 超大 (30m+) |
| 核心威力 | 中等 | 中高 | 极高 | 毁灭级 |
| 引爆方式 | 投掷/碰撞 | 延时引信 | 遥控按钮 | 倒计时 |
| 特效 | 火焰+碎片 | 橙红火球+浓烟 | 蓝白闪光+冲击环 | 白光→蘑菇云+冲击波扩散 |
| 可用模式 | 沙盒+关卡 | 沙盒+关卡 | 关卡为主 | 沙盒限定 |

## 物理破坏系统

### 爆炸冲击波流程

1. **引爆** — 冲击波球体从爆炸中心扩散，扫描范围内所有 Cannon-es 物理体
2. **力度计算** — `force = baseForce / (1 + distance² / radius²)`，近距离摧毁、远距离推动
3. **建筑碎裂** — 受击建筑将大块刚体替换为数十个小碎片刚体，各自受力飞溅
4. **车辆掀翻** — 车辆为多刚体组合（车身+四轮），冲击波对质心施加力+扭矩，产生自然翻转/滑行/碰撞
5. **粒子收尾** — 碎片落地减速后回收刚体，转换为粒子淡出消失

### 冲击力传播规则

- 距离爆炸中心越近，受到的 impulse 越大
- 轻物体（车辆、碎片）飞得远、旋转快
- 重物体（建筑核心块）位移小、碎裂成小块
- Cannon-es 质量属性 + 阻尼自然处理响应差异

### 原子弹蘑菇云

蘑菇云使用粒子系统 + 半球网格 scale 动画组合实现，同时触发屏幕震动和冲击波环从中心向外扩散。

## 场景构成

### 建筑群
6-10 栋建筑，不同高度和颜色，使用 BoxGeometry 搭建，MeshToonMaterial flat shading 渲染。

### 街道系统
网格状道路，带虚线标线。车辆作为可推动的物理物体散布其中。

### 环境装饰
树木（球体+圆柱体）、路灯、围栏、垃圾桶等，部分可被爆炸摧毁。

## UI 交互

### 屏幕布局

- **顶部状态栏** — 当前爆炸物及剩余数量、关卡目标、得分
- **中间主区域** — Three.js 等距场景渲染
- **底部工具栏** — 四种爆炸物卡片（显示数量）、引爆按钮、重置按钮

### 拖拽放置

1. 从底部工具栏按住爆炸物卡片拖拽
2. 拖到场景中时显示半透明预览，光标跟随
3. 松手时自动吸附到地面或建筑表面（基于 raycasting 检测最近的有效面）
4. 可放置多个爆炸物后统一引爆，实现连环爆炸

### 其他操作

| 操作 | 鼠标/触屏 | 键盘 |
|---|---|---|
| 选择爆炸物 | 点击工具栏卡片 | 1 2 3 4 |
| 放置炸弹 | 拖拽卡片到场景 | 按快捷键后点击场景 |
| 撤销放置 | 右键已放置的爆炸物 / 拖回工具栏 | — |
| 引爆 | 点击引爆按钮 | Space / Enter |
| 旋转视角 | 右键拖拽 | Q / E |
| 缩放 | 滚轮 | + / - |
| 重置场景 | 点击重置按钮 | R |

### 拖拽规则

- 松手时自动吸附到场景中的有效表面
- 关卡模式有数量限制，用完的爆炸物灰色显示、不可拖拽
- 沙盒模式显示无限符号
- 右键已放置的爆炸物移除并回收数量

## 游戏模式

### 沙盒模式
- 无限爆炸物
- 自由放置和引爆
- 随时重置场景
- 纯破坏快感，无时间限制

### 关卡模式
- 有限资源（指定类型和数量的爆炸物）
- 特定目标（如"摧毁红色建筑"、"用最少炸弹清空区域"）
- 通关条件判定，星级评价
- 逐步解锁新场景和爆炸物类型

## 视角控制

- 正交相机位置固定在场景斜上方（等距视角）
- 支持 90° 旋转切换观察角度（Q/E 键或右键拖拽）
- 支持缩放（滚轮或 +/- 键）
- 没有透视变形，所有平行线保持平行
