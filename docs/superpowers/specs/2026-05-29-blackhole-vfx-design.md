# 黑洞装置视觉重设计

## 概述

重写黑洞装置的视觉效果，从简单的紫色球体膨胀改为写实物理风格的三阶段效果，外加轻量屏幕扭曲实现引力透镜。

## 三阶段效果

### 阶段 1: 黑洞形成 (0 → 2s)

| 组件 | 描述 |
|------|------|
| 核心球体 | 纯黑 `MeshBasicMaterial(color=0x000000)`，0.3 → 5 半径线性增长 |
| 光晕层 | 外围深紫球体，较大、半透明，同步膨胀 |
| 吸积盘 | `TorusGeometry` 扁平环，绕 Y 轴 3 rad/s 旋转，内白→中橙→外紫红渐变，随核心同步放大 |

### 阶段 2: 粒子螺旋吸入 (0.3s → 2.5s)

| 组件 | 描述 |
|------|------|
| 螺旋粒子 | 50+ 粒子从外围沿螺旋轨道靠近核心，轨道半径缩小、速度加快。颜色：亮紫→白→暗紫（越近越亮）。接近核心时 scale.x 拉伸 |
| 建筑吸入 | 建筑进入 3m 范围后加速向核心移动，接触核心时碎裂为大块，大块再碎成粒子流沿螺旋吸入消失。每个吸入建筑 +200 分 |

### 阶段 3: 霍金辐射爆发 (2.5s → 3.0s)

| 组件 | 描述 |
|------|------|
| 白闪光 | 核心缩小→0 瞬间，纯白球体膨胀到 10 半径，0.3s 透明消失 + 屏幕全白闪 0.15s |
| 粒子抛射 | 80+ 粒子向四面八方发射，高能白粒子先射出，低能紫红粒子后射出，粒子带拖尾轨迹 |
| 物理抛射 | 向外冲击力同步施加到周围物体 |

## 轻量屏幕扭曲（引力透镜）

- 渲染场景到 `WebGLRenderTarget`
- 全屏 quad + 自定义 `ShaderMaterial`
- 仅以黑洞屏幕坐标为中心、指定半径范围内扭曲 UV
- 扭曲强度随距离衰减
- 黑洞消失后立即解除

## 文件改动

| 文件 | 改动 |
|------|------|
| `effects.ts` | `spawnBlackHoleEffect` 完全重写（3阶段动画） |
| `renderer.ts` | 增加 `WebGLRenderTarget` + 全屏 distortion quad + shader uniforms |
| `game.ts` | `updateBlackHolePhysics` 微调参数配合新动画时序 |

## 不做的

- 不引入 EffectComposer 等外部后处理库
- 不修改其他爆炸特效
- 不改变黑洞的物理吸入/抛射逻辑（仅视觉层重写）
